#   Ferramenta prática de simulação de investimentos em fundos imobiliários. 

ImobINVEST é uma planilha/simulador de investimentos focado em Fundos Imobiliários (FIIs), que ajuda o usuário a projetar patrimônio, dividendos mensais e diferentes cenários de longo prazo a partir de aportes mensais e aporte inicial.

Este projeto foi desenvolvido como parte de atividades da DIO (Digital Innovation One), com foco em prática de Excel e educação financeira.
